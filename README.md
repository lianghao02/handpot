✨ 應用程式特色

D Kurd 10 音階配置：採用手碟最經典的 D Kurd 調性，提供 Ding (D3) 加上 9 個周圍音符的完整音域。

平均分散佈局：周圍音符平均分佈，設計考量觸控手感，更容易雙手交替敲擊。

高品質模擬音色：使用 Web Audio API 合成，包含基頻、八度音、複合五度音 (Compound Fifth)，模擬真實手碟的金屬共鳴與泛音。

數字譜記錄：自動記錄您敲擊的音符數字序列，幫助您編寫、記憶和練習旋律與節奏。

雙重標籤顯示：同時顯示音符的「數字標籤」與「音名」，方便入門者對照學習。

🕹️ 使用說明

啟動音訊：首次進入頁面會看到黑色遮罩，請點擊任一處以啟動瀏覽器的音訊引擎。

開始演奏：直接點擊手碟上的音舌或中央 Ding 即可發出聲音。

記錄旋律：您敲擊的每一個音符都會以其數字標籤顯示在底部的「數字譜記錄」區塊。

清除記錄：點擊「清除 Clear」按鈕即可清空記錄區塊，開始新的節奏練習。

隱藏標籤：點擊上方的「隱藏標籤」按鈕，可以隱藏音名和數字，純粹憑直覺或聽力進行練習。

🛠️ 開發與技術

技術棧：HTML, CSS (Tailwind CSS), JavaScript (Web Audio API)

音訊引擎：使用三個振盪器 (Sine wave) 模擬手碟特有的基頻、八度泛音和複合五度泛音，提供豐富的音色層次。

部署：透過 GitHub Pages 託管。
